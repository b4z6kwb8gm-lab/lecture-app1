§1 速度の定義

平均速度・瞬間速度・時間微分の概念を問う穴埋め問題

ア平均速度の式
時刻 \(t\) から \(t+\Delta t\) の間に位置が \(x(t)\) から \(x(t+\Delta t)\) に変化したとき、
物体の平均速度は （ア） で表される。
正解： \(\dfrac{x(t+\Delta t)-x(t)}{\Delta t}\)
解説：「単位時間あたりの位置変化」が速度。分子が変位（位置の変化量）、分母が経過時間。
イ瞬間速度の数式表現
平均速度について \(\Delta t \to 0\) の極限をとる操作を数式で表すと （イ） となる。
正解： \(\lim_{\Delta t \to 0}\dfrac{x(t+\Delta t)-x(t)}{\Delta t}\)
解説：極限記号 \(\lim\) を使って「\(\Delta t\) を限りなく 0 に近づける」操作を表す。これは微分の定義そのもの。
ウ物理量の名称
\(\Delta t \to 0\) の極限をとった平均速度を、任意の時刻 \(t\) における質点の （ウ） と定義する。
正解：（瞬間）速度
解説：微小時間における平均速度の極限 → 「その瞬間の速度」＝瞬間速度。位置の時間微分 \(v=\dot{x}=dx/dt\) で表す。
エ速度が正のとき
速度（ウ）\(> 0\) のとき、物体は \(x\) 軸方向に対して （エ） 方向に進む。
正解：正
解説：速度が正 = \(x\) が増加中 = \(x\) 軸の正方向へ移動している。
オ速度が負のとき
速度（ウ）\(< 0\) のとき、物体は \(x\) 軸方向に対して （オ） 方向に進む。
正解：負
解説：速度が負 = \(x\) が減少中 = \(x\) 軸の負方向へ移動している。
カ速度がゼロのとき
速度（ウ）\(= 0\) のとき、物体は （カ） 状態である。
正解：静止
解説：速度 = 位置変化の割合 = 0 ならば、その瞬間は移動しておらず「静止」している状態。

ステップバイステップ解説：速度の定義

① 変位とは何か

位置 \(x(t)\) から \(x(t+\Delta t)\) に移動したとき、その変化量を変位と呼ぶ。

\[\Delta x = x(t+\Delta t) - x(t)\]

変位は「どれだけ動いたか」を表す量で、正・負の値を取り得る（向きの情報を持つ）。

② 平均速度の定義

1

単位時間あたりの変位＝速度の「粗い近似」。これを平均速度と呼ぶ。

\[\bar{v} = \frac{\Delta x}{\Delta t} = \frac{x(t+\Delta t)-x(t)}{\Delta t}\]
ポイント

分子は「終わりの位置 − 始まりの位置」、分母は「経過時間」。単位は [m/s]。

③ 極限と微分 — 瞬間速度へ

2

\(\Delta t\) を有限のままにすると、その区間全体の「平均」しかわからない。

3

\(\Delta t \to 0\) の極限をとると、ある瞬間の速度（瞬間速度）が得られる。

\[v(t) = \lim_{\Delta t \to 0} \frac{x(t+\Delta t)-x(t)}{\Delta t} = \frac{dx}{dt} = \dot{x}\]

これが微分の定義そのもの。「\(dx/dt\)」や「\(\dot{x}\)」（ドット記法）で表す。

④ 符号の意味

  • \(v > 0\) ：\(x\) が増加 → 正方向へ移動
  • \(v < 0\) ：\(x\) が減少 → 負方向へ移動
  • \(v = 0\) ：\(x\) が変化しない → 静止（その瞬間）
🔑 キーポイントまとめ
  • 平均速度 \(= \Delta x / \Delta t\)（区間全体の変化率）
  • 瞬間速度 \(= \lim_{\Delta t\to 0} \Delta x/\Delta t = dx/dt\)（微分）
  • 速度の符号 = 運動の向き（正 or 負方向）
  • \(v=0\) は「その瞬間の静止」（その前後で動いていても良い）

§2 加速度の定義

平均加速度・瞬間加速度・等速直線運動の概念

キ平均加速度の式
時刻 \(t\) から \(t+\Delta t\) の間に速度が \(v(t)\) から \(v(t+\Delta t)\) に変化したとき、単位時間あたりの速度変化を （キ） という。
正解： \(\dfrac{v(t+\Delta t)-v(t)}{\Delta t}\)
解説：速度の変化量 \(\Delta v\) を経過時間 \(\Delta t\) で割ったもの。
ク瞬間加速度の数式表現
平均加速度について \(\Delta t \to 0\) の極限を数式で表すと （ク） となる。
正解： \(\lim_{\Delta t \to 0}\dfrac{v(t+\Delta t)-v(t)}{\Delta t}\)
解説：速度に対して微分操作を適用する。これが瞬間加速度の定義式。
ケ物理量の名称
\(\lim_{\Delta t \to 0}\dfrac{v(t+\Delta t)-v(t)}{\Delta t}\) を、任意の時刻 \(t\) における質点の （ケ） と定義する。
正解：（瞬間）加速度
解説：速度の時間微分 \(a = dv/dt = \ddot{x}\)。位置を2回微分した量でもある。
コ正方向運動・加速度が正のとき
物体が \(x\) 軸正方向に運動しているとき、加速度（ケ）\(> 0\) ならば、この物体は （コ） している。
正解：加速
解説：速度が正、加速度も正 → 速度がさらに大きくなっている = 加速。
サ正方向運動・加速度が負のとき
物体が \(x\) 軸正方向に運動しているとき、加速度（ケ）\(< 0\) ならば、この物体は （サ） している。
正解：減速
解説：速度は正方向だが加速度が負（逆向き）→ 速度が小さくなっている = 減速。
シ加速度がゼロの運動
加速度（ケ）\(= 0\) の場合、速度は一定で運動している。この運動を （シ） と呼ぶ。
正解：等速直線運動
解説：加速度が常にゼロ = 速度が変化しない = 速さと向きが一定 = 等速直線運動。

ステップバイステップ解説：加速度の定義

① 加速度 = 速度の変化率

速度 \(v(t)\) がどれだけ速く変化するかを表す量が加速度。

\[a(t) = \lim_{\Delta t \to 0}\frac{v(t+\Delta t)-v(t)}{\Delta t} = \frac{dv}{dt} = \ddot{x}\]
ポイント：二重微分

加速度は位置 \(x\) を時間 \(t\) で 2 回微分したもの：\(a = \dfrac{d^2x}{dt^2}\)。ニュートンの記法では \(\ddot{x}\)（ドット2つ）と書く。

② 速度と加速度の符号の組み合わせ

1

正方向に進んでいる（\(v > 0\)）かつ \(a > 0\)：速度がさらに増加 → 加速

2

正方向に進んでいる（\(v > 0\)）かつ \(a < 0\)：速度が減少 → 減速

3

負方向に進んでいる（\(v < 0\)）かつ \(a < 0\)：速さ（絶対値）がさらに増加 → 加速

4

負方向に進んでいる（\(v < 0\)）かつ \(a > 0\)：速さが減少 → 減速

「加速か減速か」は速度と加速度の符号が同じか違うかで決まる。

③ 等速直線運動

\(a = 0\) ならば \(v = \text{const}\)（定数）。位置は時間の 1 次関数 \(x(t) = x_0 + v_0 t\) となる。

🔑 キーポイントまとめ
  • 平均加速度 \(= \Delta v / \Delta t\)、瞬間加速度 \(= dv/dt = d^2x/dt^2\)
  • 加速・減速の判定：\(v\) と \(a\) の符号が同じ→加速、違う→減速
  • \(a = 0\) → 等速直線運動（速度一定）
  • 単位：[m/s²]（速度 [m/s] をさらに時間 [s] で割る）

§3 位置・速度・加速度の計算

\(x(t) = 1 + 3t + 2t^2\) の微分計算

問3-1速度 \(v(t)\) を求める
位置 \(x(t) = 1 + 3t + 2t^2\) のとき、速度 \(v(t) = dx/dt\) はどれか。
正解： \(v(t) = 3 + 4t\)
解説：\(d(1)/dt = 0\)、\(d(3t)/dt = 3\)、\(d(2t^2)/dt = 4t\) → \(v = 3 + 4t\)
問3-2加速度 \(a(t)\) を求める
位置 \(x(t) = 1 + 3t + 2t^2\) のとき、加速度 \(a(t) = dv/dt\) はどれか。
正解： \(a(t) = 4\)（定数）
解説：\(v(t) = 3 + 4t\) を微分すると \(a = 4\)。時間によらず一定 → 等加速度運動。
問3-3\(v\)–\(t\) グラフの形
\(v(t) = 3 + 4t\) の \(v\)–\(t\) グラフはどのような形か。
正解：傾き 4、切片 3 の右上がりの直線
解説：\(v = 4t + 3\) は \(t\) の1次関数。\(t=0\) で \(v=3\)、時間とともに一定の割合（4 [m/s²]）で増加。
問3-4\(a\)–\(t\) グラフの形
\(a(t) = 4\) の \(a\)–\(t\) グラフはどのような形か。
正解：\(a = 4\) の水平な直線（横軸に平行）
解説：加速度が定数なので、どの時刻でも \(a=4\)。\(t\) に依存せず一定のグラフになる。

ステップバイステップ解説：多項式の微分計算

① 微分の基本公式

\[\frac{d}{dt}(t^n) = n\,t^{n-1}, \quad \frac{d}{dt}(c) = 0 \quad (c\text{ は定数})\]

② 各項の微分

与えられた位置 \(x(t) = 1 + 3t + 2t^2\) を項ごとに微分する。

1

\(\dfrac{d}{dt}(1) = 0\)（定数の微分はゼロ）

2

\(\dfrac{d}{dt}(3t) = 3 \cdot 1 \cdot t^{0} = 3\)（\(n=1\) の場合）

3

\(\dfrac{d}{dt}(2t^2) = 2 \cdot 2t^{1} = 4t\)（\(n=2\) の場合）

\[v(t) = \frac{dx}{dt} = 0 + 3 + 4t = 3 + 4t\]

③ 速度をさらに微分して加速度

4

\(\dfrac{d}{dt}(3) = 0\)（定数）

5

\(\dfrac{d}{dt}(4t) = 4\)

\[a(t) = \frac{dv}{dt} = 0 + 4 = 4 \text{ [m/s}^2\text{]}\]

④ グラフの読み方

\(v\)–\(t\) グラフ：一次関数なので直線。傾き＝加速度 \(a\)、切片＝初速度 \(v_0\)。
\(a\)–\(t\) グラフ：定数なので水平線。面積＝速度変化（\(\Delta v\)）。

ポイント：グラフと微積分の関係

\(v\)–\(t\) グラフの傾きが加速度。
\(v\)–\(t\) グラフの面積が変位（積分で求める）。
\(x\)–\(t\) グラフの傾きが速度。

🔑 キーポイントまとめ
  • \(\dfrac{d}{dt}(t^n) = nt^{n-1}\) が基本（べき関数の微分）
  • 定数の微分 = 0
  • この問題：\(x=1+3t+2t^2 \Rightarrow v=3+4t \Rightarrow a=4\)
  • \(a\) が定数 → 等加速度運動（中学・高校物理でおなじみの運動）

§4 変位ベクトル・位置ベクトル

二次元座標上の変位計算と接線方向

問4-1A→B の変位ベクトル
点A\((3,5)\) から点B\((2,8)\) に移動したときの変位ベクトルはどれか。
正解： \(\vec{AB} = (-1,\, 3)\)
解説：変位 = 終点 − 始点。\(x\) 成分：\(2-3=-1\)、\(y\) 成分：\(8-5=3\)
問4-2点C の位置ベクトル
点B\((2,8)\) から点Cへの変位ベクトルが \((7,-6)\) のとき、点Cの位置ベクトルはどれか。
正解： \(\vec{OC} = (9,\, 2)\)
解説：位置ベクトル = Bの位置 + 変位。\(x\): \(2+7=9\)、\(y\): \(8+(-6)=2\)
問4-3放物線上の変位ベクトル
物体が \(y = x^2\) の軌跡上を動き、\(x\) 座標が \(x_1\) から \(x_1 + \Delta x\) に変化したとき、変位ベクトルはどれか。
正解： \(\Delta\vec{r} = (\Delta x,\; 2x_1\Delta x + \Delta x^2)\)
解説：始点 \((x_1, x_1^2)\)、終点 \((x_1+\Delta x, (x_1+\Delta x)^2)\)。
\(\Delta y = (x_1+\Delta x)^2 - x_1^2 = 2x_1\Delta x + \Delta x^2\)
問4-4\(\Delta x \to 0\) 極限の方向
\(\Delta x \to 0\) の極限で変位ベクトルの方向角の \(\tan\) 値（傾き）はどうなるか。
正解： \(\tan\theta = 2x_1\)（\(x_1\) における接線の傾き）
解説：\(\dfrac{\Delta y}{\Delta x} = 2x_1 + \Delta x \xrightarrow{\Delta x\to 0} 2x_1 = \dfrac{d(x^2)}{dx}\bigg|_{x=x_1}\)

ステップバイステップ解説：変位ベクトルと接線

① 変位ベクトルの求め方

2次元座標で、始点 \(P_1 = (x_1, y_1)\)、終点 \(P_2 = (x_2, y_2)\) のとき：

\[\vec{P_1 P_2} = P_2 - P_1 = (x_2 - x_1,\; y_2 - y_1)\]

成分ごとに「終点 − 始点」を計算するだけ。

② 位置ベクトルの求め方

\[\vec{OC} = \vec{OB} + \vec{BC}\]

「原点からBまで移動して、さらにBからCまで変位する」イメージ。成分ごとに足す。

③ 放物線上の変位と接線

1

始点 \((x_1,\; x_1^2)\)、終点 \((x_1+\Delta x,\; (x_1+\Delta x)^2)\)

2

\(\Delta y = (x_1+\Delta x)^2 - x_1^2 = 2x_1\Delta x + (\Delta x)^2\)

3

変位ベクトル：\(\Delta\vec{r} = (\Delta x,\; 2x_1\Delta x + (\Delta x)^2)\)

4

\(\Delta x \to 0\) の極限で傾き：\(\dfrac{\Delta y}{\Delta x} \to 2x_1\)

\[\frac{d}{dx}(x^2) = 2x \implies \text{接線の傾き（} x=x_1 \text{）} = 2x_1\]
ポイント：微分の幾何学的意味

微分は「曲線の接線の傾き」を意味する。変位ベクトルの方向 \(\Delta y/\Delta x\) が \(\Delta x \to 0\) で接線方向に収束するのはそのため。

🔑 キーポイントまとめ
  • 変位ベクトル = 終点の位置ベクトル − 始点の位置ベクトル
  • 位置ベクトル = 始点の位置 + 変位
  • 曲線上の微小変位の方向 \(\to\) 接線方向（微分の幾何的意味）
  • \(y=x^2\) の接線傾き at \(x=x_1\) は \(2x_1\)

§5 内積・仕事

ベクトルの内積計算と物理への応用

問5-13次元ベクトルの内積（成分表示）
\(\vec{A} = (3,-5,1)\)、\(\vec{B} = (7,-9,1)\) のとき、内積 \(\vec{A}\cdot\vec{B}\) の値はどれか。
正解： \(67\)
解説：\(3\times7 + (-5)\times(-9) + 1\times1 = 21 + 45 + 1 = 67\)
問5-2単位ベクトル表示の内積
\(\vec{A} = 2\vec{e}_x + 4\vec{e}_y + 9\vec{e}_z\)、\(\vec{B} = -\vec{e}_x + 5\vec{e}_y + 2\vec{e}_z\) のとき、内積はどれか。（\(\vec{e}_i\) は直交単位ベクトル）
正解： \(36\)
解説：\(2\times(-1) + 4\times5 + 9\times2 = -2 + 20 + 18 = 36\)
直交座標では \(\vec{e}_x\cdot\vec{e}_x = 1\)、\(\vec{e}_x\cdot\vec{e}_y = 0\) を使う。
問5-3内積の分配法則
\(\vec{A}\cdot(\vec{B}+\vec{C}) = \vec{A}\cdot\vec{B} + \vec{A}\cdot\vec{C}\) を成分表示で示す。この証明に使う性質はどれか。
正解：実数の加法についての分配法則
解説：内積を成分表示すると \(\sum_i A_i(B_i+C_i) = \sum_i A_i B_i + \sum_i A_i C_i\)。実数の分配則から自然に成立。
問5-4仕事の計算
力 \(\vec{F} = 6\vec{e}_x + 8\vec{e}_y\) を加えて点A\((2,1)\) から点B\((4,6)\) まで動かしたときの仕事はどれか。
正解： \(52\text{ J}\)
解説：変位 \(\vec{d} = (4-2, 6-1) = (2, 5)\)。仕事 \(W = \vec{F}\cdot\vec{d} = 6\times2 + 8\times5 = 12 + 40 = 52\text{ J}\)

ステップバイステップ解説：内積と仕事

① 内積の定義（2通り）

幾何学的定義：

\[\vec{A}\cdot\vec{B} = |\vec{A}||\vec{B}|\cos\theta\]

成分表示（3次元直交座標）：

\[\vec{A}\cdot\vec{B} = A_x B_x + A_y B_y + A_z B_z\]
ポイント：直交単位ベクトルの内積

\(\vec{e}_i \cdot \vec{e}_j = \delta_{ij}\)（クロネッカーのデルタ）。つまり同じ方向は 1、異なる方向は 0。

② 分配法則の証明

\(\vec{A} = (A_x, A_y, A_z)\)、\(\vec{B} = (B_x, B_y, B_z)\)、\(\vec{C} = (C_x, C_y, C_z)\) とする。

\begin{align} \vec{A}\cdot(\vec{B}+\vec{C}) &= A_x(B_x+C_x) + A_y(B_y+C_y) + A_z(B_z+C_z)\\ &= (A_xB_x+A_yB_y+A_zB_z) + (A_xC_x+A_yC_y+A_zC_z)\\ &= \vec{A}\cdot\vec{B} + \vec{A}\cdot\vec{C} \end{align}

③ 物理における仕事

1

変位ベクトル \(\vec{d}\) を求める：\(\vec{d} = \text{B} - \text{A} = (4-2, 6-1) = (2, 5)\)

2

仕事 \(W = \vec{F}\cdot\vec{d}\)（力と変位の内積）

3

\(W = 6\times2 + 8\times5 = 12 + 40 = 52 \text{ J}\)

ポイント：仕事 = 力の「変位方向成分」× 変位の大きさ

内積 \(W = \vec{F}\cdot\vec{d} = |\vec{F}||\vec{d}|\cos\theta\)。力が変位と直角（\(\theta=90°\)）なら仕事はゼロ。

🔑 キーポイントまとめ
  • 内積 \(= \sum_i A_i B_i\)（成分の積の和）
  • 直交単位ベクトル：\(\vec{e}_i\cdot\vec{e}_j = \delta_{ij}\)
  • 分配法則：\(\vec{A}\cdot(\vec{B}+\vec{C}) = \vec{A}\cdot\vec{B}+\vec{A}\cdot\vec{C}\)
  • 仕事 \(W = \vec{F}\cdot\vec{d}\)（力と変位の内積）単位は J

§6 2次元極座標と加速度

極座標表示・速度・加速度の導出

問6-1極座標と直交座標の関係
極座標 \((r, \theta)\) と直交座標 \((x, y)\) の関係はどれか。
正解： \(x = r\cos\theta,\; y = r\sin\theta\)
解説：原点からの距離 \(r\)、偏角 \(\theta\) による三角関数の定義から直接導ける。
問6-2極座標系の単位ベクトル \(\vec{e}_r\)
直交座標の単位ベクトル \(\vec{e}_x, \vec{e}_y\) を用いて、径方向単位ベクトル \(\vec{e}_r\) を表すとどれか。
正解： \(\vec{e}_r = \cos\theta\,\vec{e}_x + \sin\theta\,\vec{e}_y\)
解説：\(\vec{e}_r\) は \(r\) の増加方向（原点から外向き）の単位ベクトル。
問6-3極座標系の単位ベクトル \(\vec{e}_\theta\)
角方向単位ベクトル \(\vec{e}_\theta\) を直交座標の単位ベクトルで表すとどれか。
正解： \(\vec{e}_\theta = -\sin\theta\,\vec{e}_x + \cos\theta\,\vec{e}_y\)
解説：\(\vec{e}_\theta\) は \(\theta\) の増加方向（反時計回り接線方向）の単位ベクトル。\(\vec{e}_r\) を \(\theta\) で微分して求められる。
問6-4極座標での速度
極座標での速度ベクトルを \(\vec{e}_r, \vec{e}_\theta\) で表すとどれか。
正解： \(\vec{v} = \dot{r}\,\vec{e}_r + r\dot{\theta}\,\vec{e}_\theta\)
解説：\(\vec{r} = r\vec{e}_r\) を時間微分。\(\dot{\vec{e}}_r = \dot{\theta}\vec{e}_\theta\) に注意して計算。
問6-5加速度の径成分
極座標での加速度 \(\vec{a}\) の径成分（\(\vec{e}_r\) 方向）はどれか。
正解： \(\ddot{r} - r\dot{\theta}^2\)
解説：\(-r\dot{\theta}^2\) は遠心力項（向心加速度の逆向き）。\(\ddot{r}\) は径方向の加速度。
問6-6加速度の角成分
極座標での加速度 \(\vec{a}\) の角成分（\(\vec{e}_\theta\) 方向）はどれか。
正解： \(2\dot{r}\dot{\theta} + r\ddot{\theta}\)
解説：\(2\dot{r}\dot{\theta}\) はコリオリ加速度、\(r\ddot{\theta}\) は角加速度に由来する項。

ステップバイステップ解説：2次元極座標

① 極座標の定義

\[x = r\cos\theta, \quad y = r\sin\theta\]

\(r = \sqrt{x^2+y^2}\)（原点からの距離）、\(\theta = \arctan(y/x)\)（偏角）。

② 単位ベクトルの導出

位置ベクトル \(\vec{r} = r\vec{e}_r\) なので：

\[\vec{e}_r = \cos\theta\,\vec{e}_x + \sin\theta\,\vec{e}_y\]

\(\vec{e}_\theta\) は \(\vec{e}_r\) に直交（反時計回り）：

\[\vec{e}_\theta = -\sin\theta\,\vec{e}_x + \cos\theta\,\vec{e}_y\]
ポイント：単位ベクトルの時間微分

\(\dot{\vec{e}}_r = \dot{\theta}\vec{e}_\theta\)、\(\dot{\vec{e}}_\theta = -\dot{\theta}\vec{e}_r\)。これが極座標計算の鍵。

③ 速度の導出

1

\(\vec{r} = r\vec{e}_r\) を時間微分：

\begin{align} \dot{\vec{r}} &= \dot{r}\vec{e}_r + r\dot{\vec{e}}_r = \dot{r}\vec{e}_r + r\dot{\theta}\vec{e}_\theta \end{align}

④ 加速度の導出

2

\(\vec{v} = \dot{r}\vec{e}_r + r\dot{\theta}\vec{e}_\theta\) をさらに時間微分：

\begin{align} \dot{\vec{v}} &= \ddot{r}\vec{e}_r + \dot{r}\dot{\theta}\vec{e}_\theta + (\dot{r}\dot{\theta} + r\ddot{\theta})\vec{e}_\theta + r\dot{\theta}(-\dot{\theta}\vec{e}_r)\\ &= (\ddot{r} - r\dot{\theta}^2)\vec{e}_r + (2\dot{r}\dot{\theta} + r\ddot{\theta})\vec{e}_\theta \end{align}

⑤ 各項の物理的意味

  • \(\ddot{r}\)：径方向の加速度
  • \(-r\dot{\theta}^2\)：向心加速度（内向き、遠心力の源）
  • \(r\ddot{\theta}\)：角加速度による接線加速度
  • \(2\dot{r}\dot{\theta}\)：コリオリ加速度（回転座標系特有）
🔑 キーポイントまとめ
  • \((x,y) = (r\cos\theta, r\sin\theta)\)
  • \(\vec{e}_r = (\cos\theta, \sin\theta)\)、\(\vec{e}_\theta = (-\sin\theta, \cos\theta)\)
  • 速度：\(\vec{v} = \dot{r}\vec{e}_r + r\dot{\theta}\vec{e}_\theta\)
  • 加速度：\(\vec{a} = (\ddot{r}-r\dot{\theta}^2)\vec{e}_r + (2\dot{r}\dot{\theta}+r\ddot{\theta})\vec{e}_\theta\)
  • 計算の核心：\(\dot{\vec{e}}_r = \dot{\theta}\vec{e}_\theta\)、\(\dot{\vec{e}}_\theta = -\dot{\theta}\vec{e}_r\)
