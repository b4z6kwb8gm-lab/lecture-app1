第一部

必要な基礎知識

ミクロな状態とマクロな状態

統計力学を理解する第一歩は、「ミクロ」と「マクロ」の違いを把握することです。

マクロな状態（巨視的状態）とは、私たちが日常的に観測できる量——温度・圧力・体積・エネルギー——で記述される状態です。例えば「この気体は 300 K で 1 気圧」というのがマクロな記述です。

ミクロな状態（微視的状態）とは、系を構成するすべての粒子一つ一つの位置・運動量（古典力学）や量子状態（量子力学）を完全に指定した状態です。同じマクロな状態でも、粒子の配置の仕方は無数にあります。

統計力学の根本的な問い：
「同じマクロな状態に対応するミクロな状態は何通りあるか？」——これが \(W(E,V,N)\) です。

小正準集合と等重率の原理

系のエネルギーが \([E,\, E+\delta E]\)（\(\delta E\) はごく小さな幅）、体積 \(V\)、粒子数 \(N\) に指定されたとき、その条件を満たすミクロな状態の総数を \(W(E,V,N)\) と書きます。

小正準分布（等重率の原理）
$$P_j = \frac{1}{W(E,V,N)}$$

「許されるすべてのミクロな状態は等しい確率で実現する」という統計力学の根本仮定。
「等重率の原理」「小正準集合」「ミクロカノニカルアンサンブル」とも呼ばれる。

なぜこれでよいのか？「典型状態」の概念

マクロな物理量 \(X\) の期待値（平均値）を計算してみましょう。

$$\langle X \rangle = \sum_j X_j P_j$$

ミクロな状態を 典型状態（マクロな物理量がほぼ同じ値 \(X\) を示す圧倒的多数の状態）と 非典型状態（数が圧倒的に少ない状態）に分けると：

$$\langle X \rangle = \sum_{j:\text{典型}} X_j P_j + \sum_{j:\text{非典型}} X_j P_j$$

非典型状態の数は圧倒的に少ないので第二項は無視でき：

$$\langle X \rangle \approx X \cdot \frac{W_\text{eq}(E,V,N)}{W(E,V,N)} \approx X$$

ここで \(W \approx W_\text{eq}\)（非典型状態の数は圧倒的少数）を使いました。つまり期待値はほぼ確実に典型値と一致します。これが統計力学がうまく機能する理由です。

格子系：\(W\) の具体的な計算例

\(N\) 個のサイトがあり、各サイトには次のどちらかが置かれます：

  • ○（エネルギー 0）
  • ◎（エネルギー \(\varepsilon\)）

問い： 全系のエネルギーが \(E = n\varepsilon\) のとき、状態数 \(W(E,N)\) は？

答え

\(E = n\varepsilon\) ということは ◎ がちょうど \(n\) 個あるということ。\(N\) 個のサイトから \(n\) 個を選ぶ組合せの数だから：

$$W(E=n\varepsilon,\, N) = \binom{N}{n} = \frac{N!}{(N-n)!\, n!}$$

エントロピーとボルツマンの原理

エントロピーは状態数の関数

熱力学の第二法則は「孤立系で変化が起きると、エントロピー \(S\) は増大（または一定）する」と述べます。これを統計力学的に読むと：

「孤立系で実現する平衡状態は \(S\) が最大となっている」
＝「平衡状態では \(W\) が最大」（問(2)の答え）
⇒ \(S\) と \(W\) は同じ方向に増減する。つまり \(S = f(W)\) と書けるはず。

関数形の決定：加法性から対数が導かれる

エントロピーは加法的でなければならない：\(S_{A+B} = S_A + S_B\)

状態数は積になる：\(W_{A+B} = W_A \cdot W_B\)

したがって \(f\) は次の関数方程式を満たす：

$$f(W_A \cdot W_B) = f(W_A) + f(W_B)$$

この方程式を \(x\)（\(=W_A\)）で偏微分すると \(y f'(xy) = f'(x)\)、\(y\)（\(=W_B\)）で偏微分すると \(x f'(xy) = f'(y)\)。これらから：

$$\frac{y}{x} = \frac{f'(x)}{f'(y)} \implies y f'(y) = x f'(x) = \text{const.} =: k_B$$ $$\therefore\quad f'(x) = \frac{k_B}{x} \implies f(x) = k_B \log x$$
ボルツマンの原理（統計力学の基本関係式）
$$S = k_B \log W_\text{eq} \approx k_B \log W$$

マクロな量 \(S\) とミクロな量 \(W\) を結ぶ橋。\(k_B \approx 1.38 \times 10^{-23}\ \text{J K}^{-1}\) はボルツマン定数。

ボルツマン定数の値の決定

理想気体を断熱膨張（体積 \(V \to 2V\)）させたときのエントロピー差を２通りで計算します：

統計力学側： \(\Delta S = N k_B \log 2\)

熱力学側： \(\Delta S = nR\log 2\)（\(n\): モル数、\(R\): 気体定数）

\(N = nN_A\) を使って比較すると：

$$k_B = \frac{R}{N_A} \approx 1.38 \times 10^{-23}\ \text{J K}^{-1}$$

第二部

復習問題の完全解説

断熱壁で囲まれた孤立系の中に、透熱壁で接触した系A・系Bがある設定です。接触面はエネルギーのやり取りが自由にできますが、外界とのやり取りはありません。

(1)
全系の状態数 \(W\) を \(W_A,\, W_B\) で表せ
系A，Bのエネルギーがそれぞれ \(E_A,\, E_B\) のとき，全系が取ることが可能な状態数 \(W\) を \(W_A\), \(W_B\) で表わせ。

考え方

系Aが取れるミクロ状態の数は \(W_A\) 通り、系Bが取れる数は \(W_B\) 通りです。系Aと系Bは独立なので、全系のミクロ状態数は各状態の組み合わせの数 = 積になります。

具体的なイメージ：
サイコロAが6通り、サイコロBが6通りなら、2つ同時に振ったときの全組み合わせは \(6 \times 6 = 36\) 通り。これと同じ論理。
答え
$$W = W_A W_B$$
(2)
平衡状態とは？
平衡状態とは，\(W\) が 　(a)　 となる状態である。(a) に適切な言葉を入れよ。

考え方

等重率の原理によれば、孤立系ではミクロには各状態が等確率で実現します。\(W\) が大きければ大きいほど、より多くのミクロ状態がその巨視的配分に対応するため、その状態に落ち着きやすくなります。系は時間発展とともに \(W\) が増大する方向に変化し、最終的に \(W\) が最大の状態——これが平衡状態——に達します。

答え

(a) 最大

平衡状態とは \(W\) が最大となる状態。言い換えると、\(S = k_B \log W\) が最大の状態。

(3)
エネルギーの関係式
系Aのエネルギーを \(E_A\)，系Bのエネルギーを \(E_B\) とし，全系のエネルギーを \(E\) とする。これらに成立する関係式を書け。

考え方

全系は孤立系なので、外界とのエネルギーのやり取りはありません。系Aと系Bの間でエネルギーのやり取りがあっても、その和は保存されます（エネルギー保存則）。

答え
$$E = E_A + E_B \quad (E \text{ は一定})$$
(4)
平衡条件の導出
平衡状態において \(S_A(E_A) := k_B \log W_A(E_A)\)，\(S_B(E_B) := k_B \log W_B(E_B)\) に成立する関係式を，(2) と (3) の条件より求めよ。

手順

ステップ1：全系のエントロピーを書く。

\(W = W_A W_B\) なので：

$$S = k_B \log W = k_B \log(W_A W_B) = \underbrace{k_B \log W_A}_{S_A} + \underbrace{k_B \log W_B}_{S_B}$$

ステップ2：\(S\) を \(E_A\)（変数）の関数として微分する。

$$dS = \frac{\partial S}{\partial E_A} dE_A + \frac{\partial S}{\partial E_B} dE_B$$

ステップ3：拘束条件 \(dE_A + dE_B = 0\)（つまり \(dE_B = -dE_A\)）を代入する。

$$dS = \left(\frac{\partial S}{\partial E_A} - \frac{\partial S}{\partial E_B}\right) dE_A$$

ステップ4：(2) より「平衡状態では \(W\)（すなわち \(S\)）が最大」なので \(dS = 0\)。\(dE_A\) は任意なので：

答え（平衡条件）
$$\frac{\partial S}{\partial E_A} = \frac{\partial S}{\partial E_B}$$

これを \(S_A, S_B\) で書き直すと：

$$\frac{\partial}{\partial E_A}(k_B \log W_A) = \frac{\partial}{\partial E_B}(k_B \log W_B)$$
この式は「平衡状態ではAとBのある量が等しくなる」ことを言っています。この量がまさに温度の逆数 \(1/T\) です（問(5)で定義します）。
(5)
熱は高温から低温へ流れることの証明
温度 \(T\) を \(\dfrac{1}{T} = \dfrac{\partial S(E)}{\partial E}\) として定義する。\(T_A > T_B\) のとき，系Aと系Bの接触後，エネルギーは系Aから系Bへ移動することを，(2) を用いて示せ。

温度の定義の確認

温度の定義を系Aと系Bそれぞれに適用すると：

$$\frac{1}{T_A} = \frac{\partial S_A}{\partial E_A}, \qquad \frac{1}{T_B} = \frac{\partial S_B}{\partial E_B}$$

証明の流れ

  1. 仮定 \(T_A > T_B\) より、\(T\) が大きいほど \(1/T\) は小さいので： $$\frac{1}{T_A} < \frac{1}{T_B} \quad \Longrightarrow \quad \frac{\partial S_A}{\partial E_A} < \frac{\partial S_B}{\partial E_B}$$
  2. 辺々引き算すると： $$0 < \frac{\partial S_B}{\partial E_B} - \frac{\partial S_A}{\partial E_A} = \frac{\partial S}{\partial E_B} - \frac{\partial S}{\partial E_A}$$
  3. 問(4)の式変形（\(dS\) を \(dE_B\) で表したもの）を使うと： $$dS = \left(-\frac{\partial S}{\partial E_A} + \frac{\partial S}{\partial E_B}\right) dE_B$$ $$\therefore \quad \frac{dS}{dE_B} = \frac{\partial S}{\partial E_B} - \frac{\partial S}{\partial E_A} > 0$$
  4. これは \(S\) が \(E_B\) の増加関数であることを意味する。(2) より平衡状態は \(S\)（\(=k_B \log W\)）が最大になる方向に系は変化するので、\(E_B\) が増える方向に変化する。
結論

\(E_B\) が増える = Bのエネルギーが増える = AからBへエネルギーが移動する

高温の系Aから低温の系Bへエネルギー（熱）が流れる。これが「熱は高温から低温へ流れる」ことの統計力学的証明。

(6)
\(k_B\) は系の種類によらない普遍定数
ボルツマンの関係式 \(S = k_B \log W\) において，エントロピーが加法的であるとすると，\(k_B\) は系の種類によらないことを論ぜよ。

論証

2つの異なる系AとBを考えます。比例定数が系によって異なる可能性を残して、それぞれ別の定数でエントロピーを書きます：

$$S_A = k_1 \log W_A, \quad S_B = k_2 \log W_B, \quad S_{A+B} = k_3 \log W_{A+B}$$

条件①（エントロピーの加法性）：

$$S_{A+B} = S_A + S_B$$

条件②（状態数の積の法則、問(1)）：

$$W_{A+B} = W_A \cdot W_B$$

これらを代入すると：

$$k_3 \log(W_A W_B) = k_1 \log W_A + k_2 \log W_B$$

左辺を展開すると：

$$k_3 \log W_A + k_3 \log W_B = k_1 \log W_A + k_2 \log W_B$$

この等式が任意の \(W_A,\, W_B\) に対して成り立つためには、\(\log W_A\) の係数と \(\log W_B\) の係数がそれぞれ等しくなければなりません：

$$k_3 = k_1 \quad \text{かつ} \quad k_3 = k_2$$
結論
$$k_1 = k_2 = k_3$$

比例定数は系の種類に依存しない普遍定数でなければならない。この定数をボルツマン定数 \(k_B\) と呼ぶ。 $$k_B = \frac{R}{N_A} \approx 1.38 \times 10^{-23}\ \text{J K}^{-1}$$

まとめ：この問題が伝えること

問題全体を通じて示されたのは次の一点です：

「平衡状態 ＝ 状態数 \(W\) が最大の状態」という等重率の原理だけから、
熱力学の主要な結果がすべて導ける。

問(1)(3)

全系の状態数は積。エネルギーは保存。

問(2)

平衡状態 ＝ \(W\)（エントロピー）が最大。

問(4)

平衡条件：\(\partial S_A/\partial E_A = \partial S_B/\partial E_B\)（＝温度が等しい）。

問(5)

熱は高温から低温へ流れる（熱力学第二法則の統計力学的導出）。

問(6)

加法性から \(k_B\) は普遍定数。\(k_B = R/N_A\)。

全体

ミクロ（等重率）⟷ マクロ（熱力学）の橋渡しが統計力学。
