1. 統計力学とは何か
なぜ統計力学が必要か？

日常で扱う物質（気体・液体・固体など）は、非常に大量の粒子からできています。

例：アボガドロ定数
$$N_A \approx 6.02 \times 10^{23} \text{ 個/mol}$$

1秒間に100万個ずつ数えたとしても、数え終わるのに約 $2 \times 10^{10}$ 年（200億年）かかります。
ちなみに宇宙の年齢は約138億年。つまり、宇宙が始まってから今まで数え続けても終わらない！

方程式は書けるが解けない

$N$個の粒子からなる系のラグランジアンは次のように書けます。

$$\mathcal{L} = \sum_{a=1}^{N} \frac{m}{2}|\dot{r}_a|^2 - \sum_{a < k} V(|r_a - r_k|)$$

第1項が全粒子の運動エネルギーの合計、第2項が粒子間のポテンシャルエネルギー（分子間ポテンシャル）です。これからラグランジュの運動方程式

$$\frac{d}{dt}\!\left(\frac{\partial \mathcal{L}}{\partial \dot{r}_a}\right) = \frac{\partial \mathcal{L}}{\partial r_a} \quad (a = 1, 2, \cdots, N)$$

が導けます。方程式の形自体は書けますが、$N \sim 10^{23}$ 個の連立方程式を解くのはスーパーコンピュータでも不可能です。解析的に解けるのは $N=2$ の2体問題のみです。

では、どうするのか？

ここが統計力学の核心的なアイデアです。「膨大さ」を逆に利用します。粒子数が多ければ多いほど、平均的な性質が安定して決まってくるという事実を使います。

直感的な例

サイコロを $N$ 回振ると、$N \to \infty$ で出た目の平均値は必ず $\dfrac{7}{2} = 3.5$ に収束します。一つ一つの目はバラバラでも、平均は決定論的に定まります。

統計力学の基本方針

「膨大さ」 $\Rightarrow$ 「確率」を用いて物理量を「決定論的」に決められる

一見「確率」と「決定論」は矛盾しているように見えますが、大数の法則によりこれが成立します。これが統計力学の出発点です。

2. 確率と統計の基礎知識
用語の定義
試行・事象・根元事象

試行：実験や観測を行う1回の操作

根元事象：1回の試行の結果（それ以上分割できない最小の結果）

事象：試行によって起こりうる結果をいくつか集めた集合（起こりえない結果を含んでいてもよい）

例：サイコロを2回振る

$E_j$ を「$j$ の目が出ること」として、$(E_j, E_k)$ は「1回目に $j$、2回目に $k$ が出ること」を表します。根元事象は $(E_1,E_1),\,(E_1,E_2),\,\ldots,\,(E_6,E_6)$ の36通りです。

標本空間と確率
標本空間 Ω

すべての根元事象を集めた集合。

  • サイコロ1回：$\Omega = \{E_1, E_2, E_3, E_4, E_5, E_6\}$
  • サイコロ2回：$\Omega = \{(E_1,E_1),(E_1,E_2),\ldots,(E_6,E_6)\}$、$|\Omega| = 36$

$\Omega$ 内のどの根元事象も同様に確からしいとき、事象 $A \subset \Omega$ の確率を

$$P(A) := \frac{|A|}{|\Omega|}$$

と定義します（$|X|$ は集合 $X$ の元の数）。

例：サイコロ2回投げ

$A$：「1回目も2回目も5以上が出る」事象

$$A = \{(E_5,E_5),\,(E_5,E_6),\,(E_6,E_5),\,(E_6,E_6)\}, \quad |A|=4$$
$$P(A) = \frac{|A|}{|\Omega|} = \frac{4}{36} = \frac{1}{9}$$
確率変数と確率分布
定義

確率変数：変数 $X$ の各値に確率が付随しているとき、$X$ を確率変数と呼ぶ

確率分布：確率変数の取る値とその確率の対応関係の全体

例：サイコロ1回投げ（X = 出た目）
$X$123456
確率 $\frac{1}{6}$$\frac{1}{6}$$\frac{1}{6}$ $\frac{1}{6}$$\frac{1}{6}$$\frac{1}{6}$

確率変数 $X$ が値 $x$ をとる確率を $x$ の関数として

$$P(X=x) = f(x) \quad \text{（}X\text{ が離散的な場合）}$$

と書きます。この $f(x)$ を確率質量関数と呼び、次の2条件を満たします。

$$\text{(i)}\; f(x) \geq 0 \qquad \text{(ii)}\; \sum_x f(x) = 1$$
連続変数の場合

$X$ が連続的な値をとるとき、確率は

$$P(a \leq X \leq b) = \int_a^b f(x)dx$$

で与えられます。$f(x)$ は確率密度関数と呼ばれ、次を満たします。

$$\text{(i)}\; f(x) \geq 0 \qquad \text{(ii)}\; \int_{-\infty}^{\infty} f(x)dx = 1$$
ガウス分布（正規分布）
$$f(x) = \frac{1}{\sqrt{2\pi\sigma^2}} \exp\!\left(-\frac{(x-\mu)^2}{2\sigma^2}\right)$$
  • $\mu$：分布の中心（平均）
  • $\sigma$：分布の広がりの目安（標準偏差）
  • $\mu \pm \sigma$ の範囲に入る確率 $\approx 0.683$（約68.3%）
3. 期待値（平均値）
期待値の定義

離散の場合：

$$E(X) = \sum_x x \cdot f(x) =: \mu$$

連続の場合：

$$E(X) := \int_{-\infty}^{\infty} x \cdot f(x)dx =: \mu$$

「各値 × その値が出る確率」をすべて足し合わせたもの。長い目で見たときの平均的な値を表します。期待値のことを平均値とも呼びます。

例：宝くじ

$E(X) \approx 150$ 円（年末ジャンボの場合）。1枚300円なので、統計的には買えば買うほど損だとわかります。

復習問題の詳細解説
問題1：穴埋め問題
問題文

熱力学とは，[ (1-1) ] な物理量の間に成立する普遍的な関係式を現象論的に考察する学問である．これに対し統計力学とは，力学や量子力学など系を構成する要素に成立する [ (1-2) ] な法則に基づき，系の [ (1-3) ] な性質を論ずる学問である．

選択肢：(a) 巨視的　(b) 微視的

解説

「巨視的（マクロ）」と「微視的（ミクロ）」の違いを理解することが鍵です。

  • 巨視的（マクロ）：温度・圧力・体積・エントロピーなど、人間が直接測定・観測できるスケールの量
  • 微視的（ミクロ）：原子・分子1個1個の位置・速度・エネルギーなど、目に見えないスケールの量

熱力学は「温度が上がれば体積が増える」といったマクロな量同士の関係式を、その背後にある原子の動きを考えずに（現象論的に）扱います。一方、統計力学はミクロな力学法則から出発し、多数の粒子が集まったときにマクロな性質がどう現れるかを導く学問です。

解答

(1-1) 巨視的　　(1-2) 微視的　　(1-3) 巨視的

問題2(1)：玉の取り出し問題
問題文

袋の中に赤玉3個・白玉2個が入っている。戻さずに2個取り出す。赤玉1個につき2点。総得点 $X$ の期待値 $E(X)$ を求めよ。

$X$ のとりうる値を列挙する

2個取り出すとき、赤玉の個数は 0, 1, 2 個のいずれかなので、

$$X = 0 \quad \text{または} \quad X = 2 \quad \text{または} \quad X = 4$$
それぞれの確率を求める

「戻さずに取り出す」ので、1回目に取り出した玉は袋に戻しません。最初に袋の中には計5個あります。

$P(X=0)$：赤玉0個 = 白玉2個を取り出す確率

  • 1回目に白玉を取る確率：$\dfrac{2}{5}$（5個中白が2個）
  • 2回目に白玉を取る確率：$\dfrac{1}{4}$（残り4個中白が1個）
$$P(X=0) = \frac{2}{5} \times \frac{1}{4} = \frac{2}{20} = \frac{1}{10}$$

$P(X=4)$：赤玉2個を取り出す確率

  • 1回目に赤玉を取る確率：$\dfrac{3}{5}$（5個中赤が3個）
  • 2回目に赤玉を取る確率：$\dfrac{2}{4}$（残り4個中赤が2個）
$$P(X=4) = \frac{3}{5} \times \frac{2}{4} = \frac{6}{20} = \frac{3}{10}$$

$P(X=2)$：赤玉1個を取り出す確率

赤玉1個・白玉1個を取り出すパターンは2通りあります：

  • パターンA：1回目が赤、2回目が白 $\to \dfrac{3}{5} \times \dfrac{2}{4} = \dfrac{6}{20}$
  • パターンB：1回目が白、2回目が赤 $\to \dfrac{2}{5} \times \dfrac{3}{4} = \dfrac{6}{20}$
$$P(X=2) = \frac{6}{20} + \frac{6}{20} = \frac{12}{20} = \frac{3}{5}$$
確率の和を確認する
$$\frac{1}{10} + \frac{3}{5} + \frac{3}{10} = \frac{1}{10} + \frac{6}{10} + \frac{3}{10} = \frac{10}{10} = 1 \quad \checkmark$$
期待値を計算する
$$E(X) = \sum_X X \cdot P(X) = 0 \times \frac{1}{10} + 2 \times \frac{3}{5} + 4 \times \frac{3}{10}$$
$$= 0 + \frac{6}{5} + \frac{12}{10} = \frac{12}{10} + \frac{12}{10} = \frac{24}{10}$$
解答
$$E(X) = \frac{12}{5} = 2.4$$
問題2(2)：ガウス分布の期待値
問題文
$$f(x) = \frac{1}{\sqrt{2\pi\sigma^2}} \exp\!\left(-\frac{(x-\mu)^2}{2\sigma^2}\right)$$

のとき、$E(X)$ を求めよ。

ヒント：$\displaystyle\int_{-\infty}^{\infty} e^{-ax^2}dx = \sqrt{\dfrac{\pi}{a}}\;(a>0)$

期待値の定義式を書く
$$E(X) = \int_{-\infty}^{\infty} x \cdot f(x)dx = \frac{1}{\sqrt{2\pi\sigma^2}} \int_{-\infty}^{\infty} x \exp\!\left(-\frac{(x-\mu)^2}{2\sigma^2}\right)dx$$
置換する

$y = x - \mu$（つまり $x = y + \mu$）とおくと、$dx = dy$、積分範囲は変わらず $-\infty$ から $\infty$ です。

$$E(X) = \frac{1}{\sqrt{2\pi\sigma^2}} \int_{-\infty}^{\infty} (y + \mu) \exp\!\left(-\frac{y^2}{2\sigma^2}\right)dy$$
積分を2つに分割する
$$E(X) = \underbrace{\frac{1}{\sqrt{2\pi\sigma^2}} \int_{-\infty}^{\infty} y \exp\!\left(-\frac{y^2}{2\sigma^2}\right)dy}_{\text{第1項}} + \underbrace{\frac{\mu}{\sqrt{2\pi\sigma^2}} \int_{-\infty}^{\infty} \exp\!\left(-\frac{y^2}{2\sigma^2}\right)dy}_{\text{第2項}}$$

第1項について：

被積分関数 $y \exp\!\left(-\dfrac{y^2}{2\sigma^2}\right)$ は奇関数です（$y$ を $-y$ に変えると符号が反転する）。奇関数を $-\infty$ から $\infty$ まで積分すると必ず0になります。

$$\int_{-\infty}^{\infty} y \exp\!\left(-\frac{y^2}{2\sigma^2}\right)dy = 0$$

第2項について：

$a = \dfrac{1}{2\sigma^2}$ とおくと、ヒントの公式より

$$\int_{-\infty}^{\infty} \exp\!\left(-\frac{y^2}{2\sigma^2}\right)dy = \sqrt{\frac{\pi}{1/(2\sigma^2)}} = \sqrt{2\pi\sigma^2}$$
$$\text{第2項} = \frac{\mu}{\sqrt{2\pi\sigma^2}} \times \sqrt{2\pi\sigma^2} = \mu$$
まとめる
$$E(X) = 0 + \mu$$
解答
$$E(X) = \mu$$
この結果の意味

ガウス分布のパラメータ $\mu$ は、そのまま分布の期待値（平均値）になっています。グラフで見てもガウス分布は $x = \mu$ を中心に左右対称なので、平均が $\mu$ になるのは直感的にも自然です。
