運動の法則 — 穴埋めクイズ

各問題で選択肢を選ぶと即座に正誤判定 ·「答えを見る」で解説表示

問題1 — 物体AとBの水平運動
設定状況の説明
物体Ａは物体Ｂの上に乗っている。Ｂの右側面には右向きの力 \(F\) が加えられている。 AとB の間、BとBの間（B底面と床の間）にはそれぞれ摩擦力が働く。 AとBは一体となって加速度 \(a\) で右向きに運動する。

Q1. 物体Ａの x 方向の運動方程式はどれか。
（AとBの間の摩擦力を \(M_1\)、Aの質量を \(m_A\)、加速度を \(a\)）

正解：\( m_A a = M_1 \)
Aに働く水平力はBとの間の摩擦力 \(M_1\) のみ（右向き）。 Fは直接Bに加わる力で、Aには伝わらない。一体運動なので加速度は同じ \(a\)。

Q2. 物体Ｂの x 方向の運動方程式はどれか。
（BとB床の間の摩擦力 \(M_2\)、Bの質量 \(m_B\)）

正解：\( m_B a = F - M_1 - M_2 \)
Bには右向き \(F\)、AからBへの反作用摩擦 \(-M_1\)（左向き）、床との摩擦 \(-M_2\)（左向き）が働く。 作用・反作用により、AがBから受ける摩擦 \(M_1\)（右）に対し、BはAから \(M_1\)（左）を受ける。

Q3. 物体Ａの y 方向の運動方程式（\(a_y=0\)）はどれか。

正解：\( 0 = N_1 - m_A g \)
垂直方向は静止（等速）なので加速度ゼロ。 垂直抗力 \(N_1\) が上向き、重力 \(m_A g\) が下向きで釣り合う。

📖 ステップバイステップ解説 — 一体運動する物体の運動方程式

① まず「どの物体に注目するか」を決める

複数の物体が登場する問題では、物体ごとに別々に運動方程式を立てるのが基本です。 「一体となって運動した」と書かれていても、それは加速度が等しいことを意味するだけで、 力は各物体に別々に働いています。

② 各物体に働く全ての力を列挙する

力の種類：

  • 重力 \(mg\)：常に鉛直下向き
  • 垂直抗力 \(N\)：接触面に垂直、面から離れる向き
  • 摩擦力 \(M\)：接触面に平行、運動を妨げる向き
  • 外力（ここでは \(F\)）：直接加えられた物体のみに働く
⚠️ ポイント
作用・反作用の法則（ニュートン第3法則）：
AがBから摩擦力 \(M_1\)（右向き）を受けるとき、BはAから同じ大きさ \(M_1\) の摩擦力を 逆向き（左向き）に受ける。符号を間違えやすいので注意。

③ 座標軸 (x, y) を設定して方程式を立てる

この問題では x を右向き正、y を上向き正に取っている。 ニュートンの第2法則 \(ma = \sum F\) を各方向に分けて書く：

\[ \text{物体A（x）:} \quad m_A a = M_1 \] \[ \text{物体A（y）:} \quad 0 = N_1 - m_A g \quad \Rightarrow \quad N_1 = m_A g \] \[ \text{物体B（x）:} \quad m_B a = F - M_1 - M_2 \] \[ \text{物体B（y）:} \quad 0 = N_2 - N_1 - m_B g \quad \Rightarrow \quad N_2 = (m_A + m_B)g \]

④ 連立して加速度・各力を求める（応用）

A の式と B の式を足すと：

\[ (m_A + m_B)\,a = F - M_2 \]

\(M_2\) は動摩擦係数 \(\mu_2\) と \(N_2\) から \(M_2 = \mu_2\,N_2 = \mu_2(m_A+m_B)g\) と表せる。 一体として扱うと全体の運動方程式に一致する。

🔑 キーポイントまとめ
  • 「一体運動」→ 加速度は同じ \(a\) だが、力の計算は各物体ごとに行う
  • 外力 \(F\) は加えられた物体（B）にのみ直接現れる
  • 作用・反作用で向きが逆になる摩擦力に要注意
  • y方向が静止 → 加速度ゼロ → 力の釣り合いと同じ
問題2 — 物体A、B、Cの運動
設定おかれている文字の定義
  • \(m_A, m_B, m_C\)：物体A、B、Cの質量
  • \(\ddot{x}, \ddot{y}\)：加速度の \(x\) 成分、\(y\) 成分
  • \(g\)：重力加速度
  • \(\theta\)：斜面の傾角
  • \(N_1\)：AとCの間の垂直抗力
  • \(N_2\)：AとBの間の垂直抗力
  • \(N_3\)：床とBの間の垂直抗力
  • \(M_1\)：AとCの間の摩擦力
  • \(M_2\)：AとBの間の摩擦力

Q1. 物体Aの \(x\) 方向の運動方程式はどれか。

正解：\( m_A\ddot{x} = N_2\sin\theta - M_2\cos\theta - M_1 \)
AはBの斜面（傾角\(\theta\)）の上に乗っている。BからAへの垂直抗力\(N_2\)の水平成分は\(N_2\sin\theta\)（正）、摩擦\(M_2\)の水平成分は\(-M_2\cos\theta\)（負）。CからAへの摩擦\(M_1\)は負（左向き）。

Q2. 物体Aの \(y\) 方向の運動方程式はどれか（\(=0\)）。

正解：\( m_A\ddot{y} = N_2\cos\theta + M_2\sin\theta - N_1 - m_Ag \quad (=0) \)
垂直抗力\(N_2\)の鉛直成分\(N_2\cos\theta\)（上向き）、摩擦\(M_2\)の鉛直成分\(M_2\sin\theta\)（上向き）、Cへの垂直抗力\(N_1\)（下向き）、重力\(m_Ag\)（下向き）。y方向は静止（加速度ゼロ）。

Q3. 物体Bの \(x\) 方向の運動方程式はどれか。

正解：\( m_B\ddot{x} = M_2\cos\theta - N_2\sin\theta \)
作用反作用の法則より、BはAから逆向きの力を受ける。摩擦の水平成分\(M_2\cos\theta\)（正）、垂直抗力の水平成分\(-N_2\sin\theta\)（負）。

Q4. 物体Bの \(y\) 方向の運動方程式はどれか（\(=0\)）。

正解：\( m_B\ddot{y} = N_3 - N_2\cos\theta - M_2\sin\theta - m_Bg \quad (=0) \)
床からの垂直抗力\(N_3\)（上）、AからBへの作用反作用で\(-N_2\cos\theta\)（下）、\(-M_2\sin\theta\)（下）、重力\(-m_Bg\)（下）。

Q5. 物体Cの \(x\) 方向の運動方程式はどれか。

正解：\( m_C\ddot{x} = M_1 \)
CはAから摩擦力\(M_1\)（右向き正）のみを水平方向に受ける。これがCの水平加速度の原因。

Q6. 物体Cの \(y\) 方向の運動方程式はどれか（\(=0\)）。

正解：\( m_C\ddot{y} = N_1 - m_Cg \quad (=0) \)
垂直抗力\(N_1\)（上向き）と重力\(m_Cg\)（下向き）が釣り合う。y方向加速度はゼロ。

📖 解説 — 物体A、B、Cの運動方程式

答え（方程式）

物体A

\[ x\text{方向}：m_A\ddot{x} = N_2\sin\theta - M_2\cos\theta - M_1 \] \[ y\text{方向}：m_A\ddot{y} = N_2\cos\theta + M_2\sin\theta - N_1 - m_Ag \quad (=0) \]

物体B

\[ x\text{方向}：m_B\ddot{x} = M_2\cos\theta - N_2\sin\theta \] \[ y\text{方向}：m_B\ddot{y} = N_3 - N_2\cos\theta - M_2\sin\theta - m_Bg \quad (=0) \]

物体C

\[ x\text{方向}：m_C\ddot{x} = M_1 \] \[ y\text{方向}：m_C\ddot{y} = N_1 - m_Cg \quad (=0) \]
🔑 キーポイント
  • 斜面（傾角\(\theta\)）からの力は水平・鉛直に分解する：垂直抗力\(N\)の水平成分\(N\sin\theta\)、鉛直成分\(N\cos\theta\)、摩擦\(M\)の水平成分\(M\cos\theta\)、鉛直成分\(M\sin\theta\)
  • 作用・反作用の法則：AがBから受ける力と、BがAから受ける力は逆向き
  • y方向静止（\(\ddot{y}=0\)）の物体は力の釣り合い式になる
問題3 — 静止している物体A、B、C
設定おかれている文字の定義
  • \(m_A, m_B, m_C\)：物体A、B、Cの質量
  • \(\ddot{x}, \ddot{y}\)：加速度の \(x\) 成分、\(y\) 成分（静止しているためすべて \(=0\)）
  • \(g\)：重力加速度
  • \(\theta\)：物体Aの傾き角
  • \(N_1\)：AとCの間の垂直抗力
  • \(N_2\)：AとBの間の垂直抗力
  • \(N_3\)：BとCの間の垂直抗力
  • \(N_4\)：床とCの間の垂直抗力
  • \(M_1\)：AとCの間の摩擦力
  • \(M_2\)：AとBの間の摩擦力
  • \(M_3\)：BとCの間の摩擦力

Q1. 物体Aの \(x\) 方向の釣り合い式はどれか（\(=0\)）。

正解：\( m_A\ddot{x} = M_1 - N_2 \quad (=0) \)
Aのx方向：Cからの摩擦\(M_1\)（右向き正）とBからの垂直抗力\(N_2\)（左向き負）が釣り合う。

Q2. 物体Aの \(y\) 方向の釣り合い式はどれか（\(=0\)）。

正解：\( m_A\ddot{y} = N_1 + M_2 - m_Ag \quad (=0) \)
Aのy方向：Cからの垂直抗力\(N_1\)（上向き）、Bからの摩擦\(M_2\)（上向き）、重力\(m_Ag\)（下向き）が釣り合う。

Q3. 物体Bの \(x\) 方向の釣り合い式はどれか（\(=0\)）。

正解：\( m_B\ddot{x} = N_2 - M_3 \quad (=0) \)
Bのx方向：Aからの垂直抗力\(N_2\)（右向き正）とCからの摩擦\(M_3\)（左向き負）が釣り合う。

Q4. 物体Bの \(y\) 方向の釣り合い式はどれか（\(=0\)）。

正解：\( m_B\ddot{y} = N_3 - M_2 - m_Bg \quad (=0) \)
Bのy方向：Cからの垂直抗力\(N_3\)（上向き）、Aからの摩擦\(M_2\)（下向き）、重力\(m_Bg\)（下向き）が釣り合う。

Q5. 物体Cの \(x\) 方向の釣り合い式はどれか（\(=0\)）。

正解：\( m_C\ddot{x} = M_3 - M_1 \quad (=0) \)
Cのx方向：Bからの摩擦\(M_3\)（右向き正）とAからの摩擦\(M_1\)（左向き負）が釣り合う。

Q6. 物体Cの \(y\) 方向の釣り合い式はどれか（\(=0\)）。

正解：\( m_C\ddot{y} = N_4 - N_1 - N_3 - m_Cg \quad (=0) \)
Cのy方向：床からの垂直抗力\(N_4\)（上向き）、Aからの垂直抗力\(N_1\)（下向き）、Bからの垂直抗力\(N_3\)（下向き）、重力\(m_Cg\)（下向き）が釣り合う。

📖 解説 — 静止している物体A、B、Cの運動方程式

答え（方程式）

物体A

\[ x\text{方向}：m_A\ddot{x} = M_1 - N_2 \quad (=0) \] \[ y\text{方向}：m_A\ddot{y} = N_1 + M_2 - m_Ag \quad (=0) \]

物体B

\[ x\text{方向}：m_B\ddot{x} = N_2 - M_3 \quad (=0) \] \[ y\text{方向}：m_B\ddot{y} = N_3 - M_2 - m_Bg \quad (=0) \]

物体C

\[ x\text{方向}：m_C\ddot{x} = M_3 - M_1 \quad (=0) \] \[ y\text{方向}：m_C\ddot{y} = N_4 - N_1 - N_3 - m_Cg \quad (=0) \]
🔑 キーポイント
  • 静止 → すべての加速度ゼロ → 各方向で合力ゼロ（力の釣り合い）
  • 各物体について接触している物体から受ける力をすべて書き出す
  • 作用・反作用：AがCから受ける力と、CがAから受ける力は逆向き
問題4 — 滑車を介した物体A、B、Cの運動
設定おかれている文字の定義
  • \(m_A, m_B, m_C\)：物体A、B、Cの質量
  • \(\ddot{x}, \ddot{y}\)：加速度の \(x\) 成分、\(y\) 成分
  • \(g\)：重力加速度
  • \(\theta\)：三角台の傾角
  • \(T\)：糸の張力
  • \(N_1\)：三角台とAの間の垂直抗力
  • \(M_1\)：三角台とAの間の摩擦力

Q1. 物体Aの \(x\) 方向の運動方程式はどれか。

正解：\( m_A\ddot{x} = T\cos\theta - M_1\cos\theta - N_1\sin\theta \)
糸の張力\(T\)の水平成分\(T\cos\theta\)（正）、摩擦\(M_1\)の水平成分\(-M_1\cos\theta\)、垂直抗力\(N_1\)の水平成分\(-N_1\sin\theta\)。

Q2. 物体Aの \(y\) 方向の運動方程式はどれか。

正解：\( m_A\ddot{y} = T\sin\theta + N_1\cos\theta - M_1\sin\theta - m_Ag \)
張力\(T\)の鉛直成分\(T\sin\theta\)（上）、垂直抗力\(N_1\)の鉛直成分\(N_1\cos\theta\)（上）、摩擦\(M_1\)の鉛直成分\(-M_1\sin\theta\)（下）、重力\(-m_Ag\)（下）。

Q3. 物体B（動滑車）の \(x\) 方向の運動方程式はどれか。

正解：\( m_B\ddot{x} = 0 \)
動滑車Bはy方向（鉛直）にのみ動く。x方向の合力はゼロ。

Q4. 物体B（動滑車）の \(y\) 方向の運動方程式はどれか。

正解：\( m_B\ddot{y} = 2T - m_Bg \)
動滑車には糸が2本かかっているため張力が\(2T\)（上向き）、重力\(m_Bg\)（下向き）。

Q5. 物体Cの \(x\) 方向の運動方程式はどれか。

正解：\( m_C\ddot{x} = 0 \)
Cは鉛直方向にのみ運動し、x方向の合力はゼロ。

Q6. 物体Cの \(y\) 方向の運動方程式はどれか。

正解：\( m_C\ddot{y} = T - m_Cg \)
Cには糸の張力\(T\)（上向き）と重力\(m_Cg\)（下向き）が働く。上向きを正とする。

📖 解説 — 滑車を介した物体A、B、Cの運動方程式

答え（方程式）

物体A

\[ x\text{方向}：m_A\ddot{x} = T\cos\theta - M_1\cos\theta - N_1\sin\theta \] \[ y\text{方向}：m_A\ddot{y} = T\sin\theta + N_1\cos\theta - M_1\sin\theta - m_Ag \]

物体B

\[ x\text{方向}：m_B\ddot{x} = 0 \] \[ y\text{方向}：m_B\ddot{y} = 2T - m_Bg \]

物体C

\[ x\text{方向}：m_C\ddot{x} = 0 \] \[ y\text{方向}：m_C\ddot{y} = T - m_Cg \]
🔑 キーポイント
  • 動滑車：糸が2本かかるため合力\(2T\)（上向き）
  • 斜面上の糸の張力は傾角\(\theta\)で水平・鉛直成分に分解する
  • 鉛直方向にしか動かない物体のx方向合力はゼロ
問題5 — 複雑な拘束をもつ物体A、B、C、Dの運動
設定おかれている文字の定義
  • \(m_A, m_B, m_C, m_D\)：物体A、B、C、Dの質量
  • \(\ddot{x}, \ddot{y}\)：加速度の \(x\) 成分、\(y\) 成分
  • \(g\)：重力加速度
  • \(\theta\)：斜面の傾角
  • \(T\)：糸の張力
  • \(N_1\)：AとB（斜面）の間の垂直抗力
  • \(N_2\)：AとCの間の垂直抗力
  • \(N_3\)：AとDの間の垂直抗力
  • \(N_4\)：床とBの間の垂直抗力
  • \(M_1\)：AとBの間の摩擦力
  • \(M_2\)：AとCの間の摩擦力
  • \(M_3\)：AとDの間の摩擦力
  • \(R_x, R_y\)：床がBを固定する力の \(x\) 成分、\(y\) 成分

Q1. 物体Aの \(x\) 方向の運動方程式はどれか。

正解：\( m_A\ddot{x} = N_1\sin\theta - M_1\cos\theta + M_2 - N_3 - T \)
Aのx方向：斜面Bからの垂直抗力の水平成分\(N_1\sin\theta\)（正）、摩擦の水平成分\(-M_1\cos\theta\)（負）、Cからの摩擦\(M_2\)（正）、Dからの垂直抗力\(-N_3\)（負）、糸の張力\(-T\)（負）。

Q2. 物体Aの \(y\) 方向の運動方程式はどれか。

正解：\( m_A\ddot{y} = N_1\cos\theta + M_1\sin\theta - N_2 - M_3 - T - m_Ag \)
Aのy方向：斜面Bからの垂直抗力の鉛直成分\(N_1\cos\theta\)（上）、摩擦の鉛直成分\(M_1\sin\theta\)（上）、Cからの垂直抗力\(-N_2\)（下）、Dからの摩擦\(-M_3\)（下）、糸\(-T\)（下）、重力\(-m_Ag\)（下）。

Q3. 物体B（床に固定）の \(x\) 方向の方程式はどれか（\(\ddot{x}=0\)）。

正解：\( m_B\ddot{x} = M_1\cos\theta - N_1\sin\theta - R_x = 0 \)
Bは床に固定されているため加速度ゼロ。Aから受ける摩擦の水平成分\(M_1\cos\theta\)（正）、垂直抗力の水平成分\(-N_1\sin\theta\)（負）、床からの固定力\(-R_x\)。

Q4. 物体B（床に固定）の \(y\) 方向の方程式はどれか（\(\ddot{y}=0\)）。

正解：\( m_B\ddot{y} = N_4 - N_1\cos\theta - M_1\sin\theta - m_Bg + R_y = 0 \)
床からの垂直抗力\(N_4\)（上）、Aからの作用反作用\(-N_1\cos\theta\)（下）、\(-M_1\sin\theta\)（下）、重力\(-m_Bg\)（下）、床の固定力\(R_y\)（上）。

Q5. 物体Cの \(x\) 方向の運動方程式はどれか。

正解：\( m_C\ddot{x} = T - M_2 \)
Cのx方向：糸の張力\(T\)（正）、Aからの摩擦\(-M_2\)（負）。

Q6. 物体Cの \(y\) 方向の運動方程式はどれか（\(=0\)）。

正解：\( m_C\ddot{y} = N_2 - m_Cg \quad (=0) \)
Cのy方向：Aからの垂直抗力\(N_2\)（上）と重力\(m_Cg\)（下）が釣り合う（y方向加速度ゼロ）。

Q7. 物体Dの \(x\) 方向の運動方程式はどれか。

正解：\( m_D\ddot{x} = N_3 - R = 0 \)
※\(R\)はAとDが離れない条件から生じる抗力など。Dのx方向は拘束によりゼロ。

Q8. 物体Dの \(y\) 方向の運動方程式はどれか。

正解：\( m_D\ddot{y} = T + M_3 - m_Dg \)
Dのy方向：糸の張力\(T\)（上向き正）、Aからの摩擦\(M_3\)（上向き正）、重力\(-m_Dg\)（下向き）。

📖 解説 — 複雑な拘束をもつ物体A、B、C、Dの運動方程式

答え（方程式）

物体A

\[ x\text{方向}：m_A\ddot{x} = N_1\sin\theta - M_1\cos\theta + M_2 - N_3 - T \] \[ y\text{方向}：m_A\ddot{y} = N_1\cos\theta + M_1\sin\theta - N_2 - M_3 - T - m_Ag \]

物体B（床に固定されているため \(\ddot{x}=0,\,\ddot{y}=0\)）

\[ x\text{方向}：m_B\ddot{x} = M_1\cos\theta - N_1\sin\theta - R_x = 0 \] \[ y\text{方向}：m_B\ddot{y} = N_4 - N_1\cos\theta - M_1\sin\theta - m_Bg + R_y = 0 \]

物体C

\[ x\text{方向}：m_C\ddot{x} = T - M_2 \] \[ y\text{方向}：m_C\ddot{y} = N_2 - m_Cg \quad (=0) \]

物体D

\[ x\text{方向}：m_D\ddot{x} = N_3 - R = 0 \quad \text{（※}R\text{はAとDが離れない条件から生じる抗力など）} \] \[ y\text{方向}：m_D\ddot{y} = T + M_3 - m_Dg \]
🔑 キーポイント
  • 物体が4つあっても各物体ごとに力を列挙してニュートン第2法則を適用する
  • 床に固定された物体B：加速度ゼロ、床からの固定力\(R_x, R_y\)が追加される
  • 糸で繋がれたC・D：張力\(T\)は同じ大きさ
  • 斜面からの力は水平・鉛直に分解して各方向の方程式に代入する
