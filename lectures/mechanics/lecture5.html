大問1空気抵抗なし
問題設定

質量 \(m\) のボールを時刻 \(t=0\) で速さ \(v_0\) で水平と角度 \(\theta\) の方向に投げ出す。初期位置は \((x,y)=(0,h)\)、空気抵抗は無視する。

0
/ 0 問正解
[小問(1)-ア]
x方向の運動方程式として正しいのはどれか？
正解：\(m\ddot{x} = 0\)
x方向には力がはたらかないので  \(m\ddot{x}=0\)、つまり等速直線運動です。
[小問(1)-イ]
y方向の運動方程式として正しいのはどれか？
正解：\(m\ddot{y} = -mg\)
重力は鉛直下向き（y負方向）なので \(m\ddot{y} = -mg\) となります。
[小問(1)-ウ]
時刻 t におけるx方向の速度 vx はどれか？
正解：\(v_x = v_0 \cos\theta\)
加速度ゼロなので \(v_x\) は初速成分 \(v_0\cos\theta\) のまま変化しません。
[小問(1)-エ]
時刻 t におけるy方向の速度 vy はどれか？
正解：\(v_y = v_0 \sin\theta - gt\)
初速y成分 \(v_0\sin\theta\) に重力積分 \(-gt\) を足します。
[小問(1)-オ]
位置 x(t) はどれか？
正解：\(x = v_0 \cos\theta \cdot t\)
vx が定数なので \(x = v_0\cos\theta \cdot t\) です（初期位置は x=0）。
[小問(1)-カ]
位置 y(t) はどれか？
正解：\(y = h + v_0\sin\theta \cdot t - \tfrac{1}{2}gt^2\)
初期位置 y=h に、初速成分と重力項を加えます。
[小問(2)-キ]
地面 y=0 に達する時刻 T は？（二次方程式の正の解）
正解：\(T = \dfrac{v_0\sin\theta + \sqrt{v_0^2\sin^2\theta + 2gh}}{g}\)
\(h + v_0\sin\theta\cdot T - \frac{1}{2}gT^2=0\) を解いて正の根を取ります。
[小問(2)-ク]
水平変位 l を最大にする tan θ はどれか？（一般式）
正解：\(\tan\theta = \dfrac{v_0}{\sqrt{v_0^2+2gh}}\)
dl/dθ = 0 を解くと \(\tan\theta = v_0/\sqrt{v_0^2+2gh}\) が得られます。
📖 ステップバイステップ解説（大問1）
STEP 1｜運動方程式を立てる

ボールに働く力は重力 \(-mg\hat{y}\) のみ（空気抵抗なし）。ニュートン第2法則より

$$m\ddot{x} = 0, \quad m\ddot{y} = -mg$$

両辺を \(m\) で割ると \(\ddot{x}=0\)、\(\ddot{y}=-g\)。

STEP 2｜速度を積分で求める

初速の各成分：\(v_{x0} = v_0\cos\theta\)、\(v_{y0} = v_0\sin\theta\)。時間積分すると

$$v_x = v_0\cos\theta, \quad v_y = v_0\sin\theta - gt$$
POINT
x方向は加速度ゼロ → 等速直線運動。y方向は等加速度 \(-g\) の運動。
STEP 3｜位置を積分で求める

速度をさらに積分し、初期条件 \(x(0)=0,\ y(0)=h\) を代入：

$$x = v_0\cos\theta \cdot t, \quad y = h + v_0\sin\theta \cdot t - \frac{1}{2}gt^2$$
STEP 4｜落下時刻 T と水平変位 l

\(y(T)=0\) とおくと \(\frac{1}{2}gT^2 - v_0\sin\theta\cdot T - h = 0\)。正の解：

$$T = \frac{v_0\sin\theta + \sqrt{v_0^2\sin^2\theta + 2gh}}{g}$$

水平変位：\(l = v_0\cos\theta \cdot T\)

STEP 5｜最適角度 θ の求め方

\(dl/d\theta = 0\) を解くと（\(s=\tan\theta\) として整理）

$$\tan\theta = \frac{v_0}{\sqrt{v_0^2+2gh}}$$
特殊ケース
\(h=0\) のとき \(\tan\theta = 1\)（\(\theta=45°\)）→ 地上から投射したときの最大飛距離角度に一致。
🗝 キーポイントまとめ
  • 空気抵抗なし → x等速・y等加速度の独立した1次元運動として解ける
  • y=0 での時刻は二次方程式の正の解；判別式に 2gh が現れる
  • 初期高さ h > 0 では最大飛距離角は 45° より小さい
  • 最適 tan θ = v₀ / √(v₀² + 2gh)
大問21次元落下＋空気抵抗
問題設定

y軸正方向を鉛直上向きとし、質量 \(m\) の物体を \(t=0\) で静かに（初速ゼロで）落下させる。空気抵抗力 \(-kv\)（\(k>0\)）が作用する。加速度と速度の時間変化グラフの概形を問う。

0
/ 0 問正解
[大問2-ア]
運動方程式として正しいのはどれか？（y軸正=上向き、落下開始）
正解：\(ma = -mg - kv\)
落下中は v<0（下向き）。重力は -mg、空気抵抗 -kv は v<0 なら正（上向き）なので \(ma=-mg-kv\)。
[大問2-イ]
初期（t=0）の加速度 a(0) はどれか？（v(0)=0）
正解：\(a(0) = -g\)
v=0 なので空気抵抗ゼロ。\(a(0) = -g\)。
[大問2-ウ]
終端速度（t→∞）v∞ はどれか？
正解：\(v_\infty = -mg/k\)
a=0 のとき \(-mg-kv=0 \Rightarrow v=-mg/k\)（下向きなので負）。
[大問2-エ]
速度 v(t) の一般解はどれか？
正解：\(v(t) = -\dfrac{mg}{k}\left(1 - e^{{-kt/m}}\right)\)
dv/dt + (k/m)v = -g を解くと \(v = -mg/k + Ce^{{-kt/m}}\)、v(0)=0 より C=mg/k。
[大問2-オ]
加速度 a(t) のグラフの正しい説明はどれか？
正解：\(t=0 で -g、時間とともに 0 に近づく（指数的）\)
\(a = -ge^{{-kt/m}}\)。t=0 で -g、指数的に 0 へ漸近。
📖 ステップバイステップ解説（大問2）
STEP 1｜運動方程式と初期条件

y正方向を上向きとすると、重力は \(-mg\)、空気抵抗は \(-kv\) です。

$$m\frac{dv}{dt} = -mg - kv, \quad v(0)=0$$
符号に注意
落下中は \(v<0\)（下向き）なので \(-kv > 0\)（上向き）。空気抵抗は落下を遅らせる。
STEP 2｜微分方程式を解く

\(\dot{v} + \frac{k}{m}v = -g\) を変数分離法で解く：

同次解 \(Ce^{-kt/m}\) ＋特殊解 \(-mg/k\)：

$$v(t) = -\frac{mg}{k} + Ce^{-kt/m}$$

\(v(0)=0\) より \(C=mg/k\)、よって

$$v(t) = -\frac{mg}{k}\left(1 - e^{-kt/m}\right)$$
STEP 3｜加速度・速度グラフの読み方

加速度：\(a(t)=\dot{v}=-ge^{-kt/m}\)

→ t=0 で \(-g\)（最大の大きさ）から指数的に 0 に漸近。

速度：t=0 で 0 から単調減少し、終端速度 \(-mg/k\) に漸近。

終端速度
\(a=0\) となる定常状態。重力と空気抵抗が釣り合い \(v_{\infty} = -mg/k\)。
🗝 キーポイントまとめ
  • 空気抵抗 -kv → 終端速度 v∞ = -mg/k が存在する
  • 速度は指数的に終端速度へ近づく（1-e^{-kt/m} の形）
  • 加速度は -g から 0 へ指数的に減衰する（e^{-kt/m} の形）
  • 時定数 τ = m/k：小さいほど速く終端に達する
大問3斜方投射＋空気抵抗
問題設定

大問1と同じ設定で、今度は空気抵抗 \(-mk\boldsymbol{v}\)（\(k>0\)）が加わる。各成分の方程式を独立に解く。

0
/ 0 問正解
[小問(1)-ア]
x方向の運動方程式はどれか？
正解：\(m\dot{v}_x = -mkv_x\)
x方向には重力なし、空気抵抗のみ。\(m\dot{v}_x = -mkv_x\)。
[小問(1)-イ]
y方向の運動方程式はどれか？
正解：\(m\dot{v}_y = -mg - mkv_y\)
y方向には重力 -mg と空気抵抗 -mkvy の両方がはたらく。
[小問(1)-ウ]
vx(t) の解はどれか？（初速 vx0 = v0 cosθ）
正解：\(v_x = v_0\cos\theta \cdot e^{{-kt}}\)
\(\dot{v}_x = -kv_x\) → \(v_x = Ce^{{-kt}}\)、初期条件より \(v_x = v_0\cos\theta\cdot e^{{-kt}}\)。
[小問(1)-エ]
vy(t) の解はどれか？（初速 vy0 = v0 sinθ）
正解：\(v_y = \left(v_0\sin\theta + \dfrac{g}{k}\right)e^{{-kt}} - \dfrac{g}{k}\)
特殊解 -g/k＋同次解 Ce^{{-kt}}。vy(0)=v0sinθ より C = v0sinθ + g/k。
[小問(1)-オ]
x(t) の解はどれか？（初期位置 x(0)=0）
正解：\(x = \dfrac{v_0\cos\theta}{k}(1 - e^{{-kt}})\)
vx を積分し x(0)=0 を使うと \(x = \frac{v_0\cos\theta}{k}(1-e^{{-kt}})\)。
[小問(2)-カ]
k≪1 のとき e^{{-kt}} ≈ 1−kt を用いると vx は？
正解：\(v_x \approx v_0\cos\theta\,(1 - kt)\)
1次近似 e^{{-kt}}≈1-kt を代入するだけ。
[小問(2)-キ]
k≪1 の1次近似で vy は？（主要な形）
正解：\(v_y \approx v_0\sin\theta - gt + k\left(\frac{g}{2}t^2 - v_0\sin\theta\cdot t\right)\)
vy 式に e^{{-kt}}≈1-kt を代入して整理すると 0次項 v0sinθ-gt に k の1次補正 k(gt²/2 - v0sinθ·t) が加わります。
[小問(3)-ク]
k≪1 の2次近似 e^{{-kt}}≈1-kt+(kt)²/2 で x(t) はどれか？
正解：\(x \approx v_0\cos\theta\left(t - \dfrac{kt^2}{2}\right)\)
x = (v0cosθ/k)(1-e^{{-kt}}) に 1-e^{{-kt}}≈kt-(kt)²/2 を代入して k で割ると v0cosθ(t - kt²/2)。
📖 ステップバイステップ解説（大問3）
STEP 1｜運動方程式（各成分）

空気抵抗 \(-mk\boldsymbol{v}\) は x・y 成分に独立にはたらく：

$$\dot{v}_x = -kv_x, \quad \dot{v}_y = -g - kv_y$$
STEP 2｜x方向の解
$$v_x = v_0\cos\theta \cdot e^{-kt}$$
$$x = \frac{v_0\cos\theta}{k}\left(1-e^{-kt}\right)$$
POINT
\(t\to\infty\) で \(v_x\to 0\)、\(x\to v_0\cos\theta/k\)（有限距離で止まる）。
STEP 3｜y方向の解

\(\dot{v}_y + kv_y = -g\) は1階線形ODE。特殊解 \(-g/k\)＋同次解 \(Ce^{-kt}\)：

$$v_y = \left(v_0\sin\theta + \frac{g}{k}\right)e^{-kt} - \frac{g}{k}$$

積分して \(y(0)=h\) を使うと：

$$y = h - \frac{g}{k}t - \frac{1}{k}\left(v_0\sin\theta + \frac{g}{k}\right)\left(e^{-kt}-1\right)$$
STEP 4｜マクローリン展開による近似（k≪1）

1次近似（\(e^{-kt} \approx 1 - kt\)）：

$$v_x \approx v_0\cos\theta\,(1-kt)$$
$$v_y \approx v_0\sin\theta - gt + k\left(\frac{g}{2}t^2 - v_0\sin\theta\cdot t\right)$$

2次近似（\(e^{-kt} \approx 1 - kt + \frac{k^2t^2}{2}\)）：

$$x \approx v_0\cos\theta\left(t - \frac{kt^2}{2}\right)$$
$$y \approx h + v_0\sin\theta\cdot t - \frac{g}{2}t^2 - kv_0\sin\theta \cdot t^2 + \frac{kg}{6}t^3 + O(k^2)$$
なぜ位置は2次まで？
位置は速度を積分するため、1次の速度誤差が \(t^2\) オーダーの位置誤差になる。精度維持のため2次展開が必要。
🗝 キーポイントまとめ
  • 速度比例の空気抵抗 → x・y独立の1階線形ODE として解ける
  • x方向：指数減衰 e^{-kt}、水平距離は v₀cosθ/k で飽和
  • y方向：終端速度 -g/k に向かう指数収束
  • k≪1 のマクローリン展開で大問1（空気抵抗なし）の結果に一致する
  • 速度は1次、位置は2次まで展開することで精度が保たれる
